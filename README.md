# SearchTab

像 macOS 聚焦搜索（Spotlight）一样，在浏览器已打开的标签页之间极速检索和切换。

**⚠️ 声明：本项目所有代码（包括实现、调试、重构、代码审查）均由 AI 大模型自动生成，未经人工 review。如有疑问请直接关闭页面，勿浪费彼此时间。**

## 功能

### 标签页搜索

- `Ctrl+Shift+Space`（macOS：`⌘+Shift+Space`）在任意网页唤起搜索面板
- 搜索已打开标签页的**标题、URL、域名、页面正文**
- 中英文混合输入、子串匹配、轻量模糊搜索
- 键盘操作：`↑` `↓` 选择、`Enter` 跳转、`Esc` 关闭
- 触控板/鼠标滚轮滚动浏览结果列表
- 自动过滤浏览器内部页面（newtab、设置、扩展管理等），仅索引互联网页面
- 输入法（IME）组合态兼容，选词时 Enter/Esc 不会误触发跳转或关闭

### 标签页快速切换

| 命令 | Windows/Linux | macOS |
|---|---|---|
| 上一个标签页 | `Ctrl+Shift+←` | `⌘+Shift+←` |
| 下一个标签页 | `Ctrl+Shift+→` | `⌘+Shift+→` |

只切换当前窗口中的互联网页面，到达边界时循环。

### 设置页面

- 点击工具栏图标 → **设置**，在独立标签页中打开
- 查看/修改所有快捷键（跳转至浏览器快捷键管理页）
- **页面正文搜索**开关：关闭后仅搜索标题/URL/域名，性能更佳
- 修改后立即生效

### 引导页

- 首次安装自动弹出，介绍使用方法和检索能力
- 工具栏图标 → **关于** 可随时重新打开

## 技术栈

WXT · React · TypeScript · MiniSearch · MV3

## 兼容性

Chrome / Edge / 所有 Chromium 内核浏览器（MV3）

## 开发

```bash
npm install
npm run dev      # 开发模式
npm run build    # 生产构建
npm run zip      # 打包 zip
```

## 加载到浏览器

1. 打开 `chrome://extensions` 或 `edge://extensions`
2. 开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `output/chrome-mv3` 目录

## 隐私

所有标签页数据仅在本地内存中建立索引，不会上传到任何服务器。关闭标签页后相关索引自动清除。
